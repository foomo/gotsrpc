## Installation

Follow the docs:

https://www.foomo.org/docs/projects/gotsrpc/cli#installation

From source to /usr/local/bin/gotsrpc:

```bash
go get github.com/foomo/gotsrpc/v2
cd $GOPATH/src/github.com/foomo/gotsrpc
make install
```
